# Print methods

Print methods

## Usage

``` r
# S3 method for class 'loo'
print(x, digits = 1, ...)

# S3 method for class 'waic'
print(x, digits = 1, ...)

# S3 method for class 'psis_loo'
print(x, digits = 1, plot_k = FALSE, ...)

# S3 method for class 'importance_sampling_loo'
print(x, digits = 1, plot_k = FALSE, ...)

# S3 method for class 'psis_loo_ap'
print(x, digits = 1, plot_k = FALSE, ...)

# S3 method for class 'psis'
print(x, digits = 1, plot_k = FALSE, ...)

# S3 method for class 'importance_sampling'
print(x, digits = 1, plot_k = FALSE, ...)
```

## Arguments

- x:

  An object returned by
  [`loo()`](https://mc-stan.org/loo/reference/loo.md),
  [`psis()`](https://mc-stan.org/loo/reference/psis.md), or
  [`waic()`](https://mc-stan.org/loo/reference/waic.md).

- digits:

  An integer passed to
  [`base::round()`](https://rdrr.io/r/base/Round.html).

- ...:

  Arguments passed to
  [`plot.psis_loo()`](https://mc-stan.org/loo/reference/pareto-k-diagnostic.md)
  if `plot_k` is `TRUE`.

- plot_k:

  Logical. If `TRUE` the estimates of the Pareto shape parameter \\k\\
  are plotted. Ignored if `x` was generated by
  [`waic()`](https://mc-stan.org/loo/reference/waic.md). To just plot
  \\k\\ without printing use the
  [plot()](https://mc-stan.org/loo/reference/pareto-k-diagnostic.md)
  method for 'loo' objects.

## Value

`x`, invisibly.
